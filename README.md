# Socket Chat

Aplicação de chat com múltiplas salas usando Node.js e Socket.IO.

São 4 salas disponíveis para conversa. As mensagens são trocadas apenas entre usuários da mesma sala.

---

## Requisitos

- Ter o Node.js instalado

---

## Como executar

1. Rode o comando:
   ```
   npm install
   ```

2. Inicie o servidor:
   ```
   npm start
   ```

3. Acesse no navegador:
   ```
   http://localhost:3000
   ```

4. Abra várias abas no navegador, escolha uma sala em cada aba e envie mensagens.

---

✅ Apenas as abas conectadas à mesma sala recebem as mensagens.
